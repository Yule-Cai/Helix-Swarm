Helix-Swarm

Helix-Swarm is a local-first CLI agent framework for LM Studio and OpenAI-compatible models. It focuses on safe tool use, local file reading, Skill/SOP integration, bilingual CLI output, and evidence-based code review.

Local-first

Built for LM Studio and OpenAI-compatible local endpoints.

Permission-gated

Low-risk reads can run automatically; installs, edits, deletes, and terminal commands require review.

Evidence Cards

Review tasks must cite concrete files, symbols, evidence, consequences, and suggested fixes.

Bilingual CLI

Switch between Chinese and English using /set lang zh or /set lang en.
