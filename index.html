// 02   capabilities

Built different.

⚡

True Parallelism

Tasks with no mutual dependencies run simultaneously via ThreadPoolExecutor. searcher + terminal at the same time — not one after another.

ThreadPoolExecutor
🔄

Dynamic Re-planning

When consecutive failures exceed a threshold, the Planner regenerates steps for remaining work. SyntaxError fast-path skips LLM entirely.

Auto-recovery
🧠

MEMORY.md

Each project distills lessons into a pointer file after every task. Injected into every Agent's header on the next run. Inspired by Claude Code.

Persistent memory
✨

autoDream

When learning stops, the system consolidates knowledge into GLOBAL_MEMORY.md, resolves contradictions, and updates pointers autonomously.

Knowledge distillation
👤

User Modeling

Automatically extracts your tech stack, style, and patterns from conversations. Injects a compact profile into every Agent — the longer you use it, the better it knows you.

Honcho-style
📚

Skill Library

Successful tasks become reusable skills. Level 0 index is always in memory (~25 tok/skill). Level 1 loads on-demand. Token cost stays flat at 100+ skills.

Progressive disclosure
⛔

Hard Cancel

Cancel stops inference mid-call — not just the frontend stream. The signal goes directly into the LLM client, killing the request immediately.

Instant stop
📱

QQ Gateway

Connect via NapCatQQ / OneBot v11. DM the bot and Helix replies with the result. Works in private chat and group chat.

OneBot v11
🌐

Full i18n

Complete Chinese / English switching — UI text, Agent instructions, and LLM system prompts all follow the selected language setting.

zh / en
